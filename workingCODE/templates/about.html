    • About the Project

    This project is a python-flask tool designed to check URLs and domains for signs of maliciousness through the use of various APIs/Open source tools used by security professionals in the cybersec industry.

    Instead of crosschecking multiple various websites you can use this tool to do a rapid check all at once, simply enter a URL or a domain on the home page and see results

    See the github at the bottom of the page for more technical information about how this project works on the backend or at: here


    APIS used for the project:

    Virus Total is an online service that analyzes suspicious files and URLs to detect types of malware and malicious content using antivirus engines and website scanners VT link

    OTX AlientVault (from AT&T) provides open access to a global community of threat researchers and security professionals. OTX AlienVault link

    Google Safe Browsing is a service developed by Google that aims to protect users across the internet from phishing attacks, malware, and other harmful content. Its service works well and offers a wide range of APIs and other integration methods for tools just like this Google SafeBrowse

    MX records is a simple DNS lookup, MX records or mail exchange records show whether or not a domain can send emails, alot of malicious domains are ones that send spam through email, if a domain is seen by malicious by the other tools+has MX records than it can be a sign of a potential email threat
